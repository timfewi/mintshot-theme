# Changelog

## 1.0.35

- Redesigned the light-theme palette system so the light themes use neutral light-first surfaces instead of carrying over dark-theme mint styling.
- Reworked `MintShot Light` and `MintShot Light Blue` with fully refreshed editor, UI, semantic token, and TextMate token colors.
- Added five new light themes: `MintShot Light Gray`, `MintShot Light Gold`, `MintShot Light Rose`, `MintShot Light Lavender`, and `MintShot Light Sand`.
- Kept italic styling for parameters and readonly variables across the redesigned light-theme set.

## 1.0.34

- Added `MintShot Light`, a mint-accented light theme with italic parameters and readonly variables.
- Added `MintShot Light Blue`, a cool blue light variant with matching italic token styling.

## 1.0.0

- Initial MintShot theme release.
- Added public repo metadata and packaging/publishing workflow.
